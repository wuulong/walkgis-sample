# 小洞天 (小洞天米腸香腸專賣店)

清大夜市的排隊名店，專賣米腸與香腸。特色是將香腸與米腸先炸後烤，表皮酥脆，並附上清爽的醃蘿蔔、泡菜、蒜頭與嫩薑解膩。點餐滿額通常會贈送四樣配菜。

### 資訊
*   **地址**: 新竹市東區建功路27號
*   **營業時間**: 平日 14:00–23:00 (週末可能公休或營業時間浮動)
*   **推薦菜色**: 米腸+香腸綜合、四樣配菜(醃蘿蔔/泡菜/蒜頭/嫩薑)
